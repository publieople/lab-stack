# 计算机基础

## 📋 概述

计算机是能够按照程序自动处理数据的电子设备。理解计算机组成原理是掌握计算机科学的基础。

## 🏗️ 计算机组成原理

### 1. 冯·诺依曼体系结构

现代计算机大多基于冯·诺依曼体系结构，包含五个核心部件：

1. **运算器 (ALU)** - 执行算术和逻辑运算
2. **控制器 (CU)** - 指挥协调各部件工作
3. **存储器 (Memory)** - 存储程序和数据
4. **输入设备** - 接收外部信息
5. **输出设备** - 输出处理结果

**💡 核心思想**：程序和数据统一存储，按顺序执行。

### 2. 硬件组成详解

#### 🧠 中央处理器 (CPU)

- **功能**：计算机的大脑，执行指令
- **组成**：
  - 运算器 (ALU)：数学计算
  - 控制器 (CU)：指令解码和控制
  - 寄存器：高速临时存储
  - 缓存：CPU 内部高速存储器

#### 💾 存储器层次结构

```graph
寄存器 → 缓存 → 主存 → 外存
  最快     次快    中等     最慢
  最小     较小    较大     最大
```

**⚡ 原理**：越靠近 CPU 的存储器速度越快，容量越小，成本越高。

#### 🔌 主板与总线

- **主板**：连接所有硬件的平台
- **总线**：数据传输通道
  - 数据总线：传输数据
  - 地址总线：指定内存位置
  - 控制总线：传输控制信号

#### ⌨️ 输入输出系统

- **输入设备**：键盘、鼠标、摄像头等
- **输出设备**：显示器、打印机、音响等
- **接口**：USB、HDMI、雷电等连接标准

### 3. 数据表示与存储

#### 🔢 二进制基础

- 计算机使用二进制（0 和 1）表示所有数据
- 位 (bit)：最小单位，0 或 1
- 字节 (byte)：8 位，基本存储单位

#### 📊 常见数据单位

```graph
1 KB = 1024 bytes
1 MB = 1024 KB
1 GB = 1024 MB
1 TB = 1024 GB
```

#### 🔣 数值表示

- 整数：原码、反码、补码
- 浮点数：IEEE 754 标准
- 字符：ASCII、Unicode 编码

### 4. 指令执行过程

#### 🔄 指令周期

1. **取指**：从内存读取指令
2. **译码**：解析指令含义
3. **执行**：执行指令操作
4. **访存**：访问内存（如果需要）
5. **写回**：将结果写回寄存器

#### 🚀 流水线技术

将指令执行过程分解为多个阶段，同时处理多条指令，提高效率。

### 5. 存储系统

#### 🧠 RAM (随机存取存储器)

- 特点：易失性，断电数据丢失
- 类型：DRAM（动态）、SRAM（静态）

#### 💿 ROM (只读存储器)

- 特点：非易失性，存储固件/BIOS
- 类型：PROM、EPROM、EEPROM、Flash

#### 💽 硬盘与固态硬盘

- **机械硬盘 (HDD)**：磁性存储，机械结构，速度较慢
- **固态硬盘 (SSD)**：闪存存储，无机械部件，速度快

### 6. 性能指标

#### 📈 关键参数

- **时钟频率**：CPU 工作节奏，单位 Hz
- **核心数**：CPU 并行处理能力
- **缓存大小**：影响数据处理速度
- **内存容量**：决定可同时运行的程序数量
- **带宽**：数据传输速率

#### ⚖️ 性能平衡

计算机性能受限于最慢的部件（木桶原理），需要各部件协调工作。

## 🛠️ 实际应用建议

### 🛒 选购计算机考虑因素

1. **用途决定配置**：办公、游戏、设计需求不同
2. **CPU 选择**：核心数、频率、架构
3. **内存容量**：至少 8GB，推荐 16GB 以上
4. **存储类型**：SSD 提升系统响应速度
5. **显卡需求**：集成显卡 vs 独立显卡

### 🔧 维护与优化

1. **散热管理**：确保良好通风
2. **定期清理**：物理灰尘和软件垃圾
3. **驱动更新**：保持硬件最佳性能
4. **备份重要数据**：防止数据丢失
