# 第三周教程：现代化UI库和工具

## 学习目标

- 掌握Tailwind CSS高级配置和自定义
- 学会使用shadcn-vue组件库
- 了解magicui等现代化UI库
- 掌握多UI库集成技巧
- 培养评估和选择工具的能力
- 提高查阅官方文档的技能

---

## 第1-3天：Tailwind CSS深度应用

### 学习内容

1. **Tailwind配置文件详解**
   - `tailwind.config.js`文件结构
   - 主题配置（theme）
   - 屏断点配置（screens）
   - 自定义插件

2. **高级配置技巧**
   - 使用@layer指令
   - 变体（variants）配置

3. **自定义组件和工具类**
   - 使用@apply指令
   - 创建可复用组件类
   - 响应式设计高级技巧

### 实践任务

- 创建一个包含多个可复用组件的Tailwind组件库
- 包括按钮、卡片、表单元素、导航组件
- 使用@apply指令创建可复用的组件类
- 实现深色模式切换功能

### 文档查阅任务

- 深入阅读Tailwind CSS官方文档的"Customizing"章节
- 学习如何创建自定义插件
- 查找并实现一个实用的Tailwind插件

---

## 第4-5天：shadcn-vue学习

### 学习内容

1. **shadcn-vue安装与配置**
   - 项目初始化配置
   - 组件安装命令

2. **常用组件使用**
   - 基础组件（Button, Input, Label）
   - 容器组件（Card, Sheet, Dialog）
   - 表单组件（Form, Select, Checkbox）

3. **主题定制**
   - 颜色主题
   - 间距和圆角
   - 动画效果

### 实践任务

- 为你的Vue应用集成shadcn-vue
- 使用至少5种不同的shadcn-vue组件
- 实现一个包含表单、对话框和数据表格的管理界面
- 自定义组件样式以匹配应用主题

### 文档查阅任务

- 访问shadcn-vue文档（或相应库的官方文档）
- 学习如何自定义组件的样式
- 查找并实现一个复杂的组件组合（如带验证的表单）

---

## 第6-7天：magicui等其他UI库

### 学习内容

1. **UI库评估标准**
   - 文档完整性
   - GitHub活跃度
   - 社区支持
   - 与Vue的兼容性
   - 性能影响

2. **集成策略**
   - 多库共存考虑
   - 冲突解决
   - 性能影响评估

### 实践任务

- 自主选择一个现代化UI库（如magicui、Framer Motion Vue版、或其他流行库）
- 评估该库的适用性
- 在你的Vue项目中集成该库
- 使用该库的特色功能增强你的应用

### 自主学习任务

- 比较至少3个不同的UI库（如shadcn-vue、Headless UI、或其他）
- 分析它们的优缺点
- 为不同场景选择最适合的库
- 准备一个简短的比较报告

### 文档查阅任务

- 选择一个你感兴趣的新UI库
- 详细阅读其官方文档
- 了解其设计理念和最佳实践
- 在你的项目中应用该库的核心功能
